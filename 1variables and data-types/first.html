Variables

Var: variables can be re-declared and updated. A global scope variable

let: Variables connot be re-declared but can be updated.

const: Variables cannot be re-declared or updated.

Data types

There are two types of data types

  • Primitive
  • Non-primitive

Primitive data types

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • BigInt
  • Symbol

Non-primitive data types

  • Object :- An object is a collection of key-value pairs.
  • Array
  • Function
