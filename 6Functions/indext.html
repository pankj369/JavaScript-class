Functions

  1. Function :- is a block of code that performs a specific task, can be invoked or called whenever needed.

  2. It helps us to repeat a task multiple times (redundancy).

  3. Function Parameters :- are like local variables of functions and it has a block scope variable.

  4. Arrow Functions :- is a compact way of writing a function.

  5. ForEach Loop in Arrays
    i. CallbackFunction :- Here, It is a function to execute for eacth element in the array.
    ii. A callback is a function passed as an argument to another function.
    iii. forEach function with callback function becomes higher order functions or methods.
    iv. forEach function is a higher order function.
    v. forEach function is a built-in function in JavaScript.

  6. Map :- Creates a new array with the results of some operation. The value its callback returns are used to form new array.

  7. Filter :- Creates a new array with those values that pass the test implemented by the callback function.

  8. Reduce :- Accumulates the elements of an array into a single value
